## 📋 测试前准备

### 1. 重置配置（用于 TC-01）

```bash
# 备份当前配置
cp "~/Library/Application Support/DataGuardScanner/config.json" \
   "~/Library/Application Support/DataGuardScanner/config.json.backup"

# 删除配置（测试默认值）
rm "~/Library/Application Support/DataGuardScanner/config.json"

# 重启应用
npm run dev
```

### 2. 准备测试数据

创建一个测试文件夹，包含以下文件：

```
测试数据/
├── test_phone.txt          # 包含手机号：18900786699
├── test_idcard.txt         # 包含身份证号：110101199001011234
├── test_password.txt       # 包含密码：password=admin123
├── test_expression.txt     # 同时包含：密码 和 身份证
└── test_mixed.docx         # Office 文档，包含多种敏感信息
```

**文件内容示例：**

**test_phone.txt:**
```
联系人：张三
电话：18900786699
邮箱：zhangsan@example.com
```

**test_idcard.txt:**
```
姓名：李四
身份证号：110101199001011234
地址：北京市朝阳区建国路100号
```

**test_password.txt:**
```
系统配置：
username=admin
password=admin123
database=mysql
```

**test_expression.txt:**
```
重要提示：
您的密码已重置，新密码为：Admin@2024
请妥善保管您的身份证号码：110101199001011234
```
