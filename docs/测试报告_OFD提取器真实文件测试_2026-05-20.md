## 💡 发现的问题与建议

### 问题 1: 包含 SVG 路径指令

**现象**:
```
M 0.27 0 L 0.27 47
M 0 0.125 L 73 0.125
...
```

**影响**:
- 对敏感词检测影响很小（这些是坐标指令，不包含敏感词）
- 会增加文本长度，但不会误报

**优化建议**（可选）:
```typescript
// 在 extractTextFromXml 方法中添加过滤
const filteredChunks = textChunks.filter(chunk => {
  // 过滤掉纯 SVG 路径指令
  return !/^M\s+\d/.test(chunk.trim());
});
```

**优先级**: 🟡 低（当前不影响功能）

---

### 问题 2: extractorName 为 undefined

**现象**:
```
🏷️  提取器: undefined
```

**原因**:
- `buildResult` 方法可能未正确设置 `extractorName`
- 或者编译后的代码中配置未正确传递

**影响**:
- 仅影响日志输出
- 不影响实际功能

**修复建议**:
检查 `BaseExtractor.buildResult` 方法的实现，确保 `this.config.name` 正确传递。

**优先级**: 🟢 中（影响日志可读性）
